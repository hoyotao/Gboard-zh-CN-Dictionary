# Gboard特殊符号中文字典 (Gboard Chinese Dictionary)

这是一个专为 Gboard (Google 键盘) 中文拼音用户设计的增强字典。

它解决了 Gboard 中文模式下特殊符号输入困难的问题，让你可以像使用搜狗/百度输入法一样，通过拼音快速打出各种符号。

## ✨ 特性 (Features)

* **覆盖全面**：包含约 1000+ 个常用符号。
    * 希腊字母 (α, β, Ω...)
    * 数学与集合符号 (∑, ∫, ∪, ⊥...)
    * 常用箭头 (→, ➽, ↺...)
    * 货币符号 (₿, ฿, ₪...)
    * IPA 英语音标 (ə, æ, ʃ...)
    * CJK 特殊字符与生僻字结构 (々, ㊊, ⿰, ⿱...)
* **四维触发逻辑**：大部分符号支持 4 种输入方式，不再为了记快捷键而头秃。
    * **全拼**：输入 `weifen` → ∂
    * **缩写**：输入 `wf` → ∂
    * **首字**：输入 `wei` → ∂
    * **语义/通用词**：输入 `fu` (符号) → ∂

## 🚀 如何安装 (How to Import)

1.  下载本项目的 `.zip` 压缩包到手机。
2.  打开 Gboard **设置 (Settings)** -> **词典 (Dictionary)**。
3.  选择 **个人词典 (Personal dictionary)** -> **中文 (Chinese)**。
4.  点击右上角的三个点 **...** -> **导入 (Import)**。
5.  选择刚才下载的 `.zip` 文件。
6.  等待显示 "Import finished" 即可。
